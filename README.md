# auto_login
대원샵, 소프라노몰 출석체크하기 귀찮아서 만들었음   
   
쓸사람 없겠지만 쓰려면 auto_login 디렉토리에 .env 파일 만들고   
   
SLACK_BOT_OAUTH_TOKEN="슬랙OAUTH키"   
SLACK_CHANNEL="#푸쉬받을슬랙채널명"   
DAEWON_ID="아이디"   
DAEWON_PW="비번"   
SOFRANO_ID="아이디"   
SOFRANO_PW="비번"   
APPLE_ID="아이디"   
APPLE_PW="비번"   
   
써놓고 저장.   
   
슬랙으로 푸쉬 안받고싶거나 애플 미리알림 체크 필요없는경우에 대해선 추후업뎃
